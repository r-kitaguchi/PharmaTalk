# Pharma Talk
<img width="1386" alt="スクリーンショット 2021-08-12 9 42 56" src="https://user-images.githubusercontent.com/83751975/129121306-7ad7190c-604a-4792-9fb8-6b58cf4856ac.png">

薬剤師と薬学生を繋ぐチャットアプリです。
薬学生は、現役薬剤師に就職に関する相談をすることができます。

## URL
https://glacial-shore-30562.herokuapp.com/

## このアプリを作成した目的
国家試験の勉強や卒業研究などで忙しい薬学生が、効率的に就活を進められるようなものが作りたいと思い、作成しました。

## 利用方法
### 薬剤師
1. 新規登録、プロフィール登録を行います。
2. マイページで学生からのトーク申請を承認し、トーク画面に移るとメッセージを送れます。(１度に承認できる人数は３人までです)
<img width="721" alt="スクリーンショット 2021-08-12 9 58 09" src="https://user-images.githubusercontent.com/83751975/129122274-9ce30ea7-b43a-49a7-8502-8e82a4b6efe2.png">
3. 相手から来たメッセージを見ていない時、マイページ(トーク中の欄)に赤いマークが付きます。
4. "トークを終了"をクリックすると、トーク履歴、承認状態が削除されます。
<img width="755" alt="スクリーンショット 2021-08-12 10 37 18" src="https://user-images.githubusercontent.com/83751975/129124921-ce85fb42-73dd-4d9b-a2df-24e54c04d49a.png">




### 学生
1. 新規登録、プロフィール登録を行います。
2. ホーム画面から薬剤師を検索し、薬剤師の詳細画面からトーク申請を行います。(１度に申請できる人数は３人までです)
<img width="810" alt="スクリーンショット 2021-08-12 9 50 31" src="https://user-images.githubusercontent.com/83751975/129121770-1b9ea535-1baf-46e9-9f95-563882bc8d7d.png">

3. トークが承認されると、トーク画面からメッセージを送れます。
4. 相手から来たメッセージを見ていない時、マイページ(トーク中の欄)に赤いマークが付きます。
5. "トークを終了"をクリックすると、トーク履歴、承認状態が削除されます。

### テストユーザーについて
薬剤師、学生ともにテストユーザーを用意しております。
ログイン画面の "テストユーザーでログイン" をクリックすると、ログインできます。

## 使用技術

- メッセージ送信、通知機能
